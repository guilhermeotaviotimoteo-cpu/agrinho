# 🌾 AgroForte - Futuro Sustentável

> O Equilíbrio Perfeito Entre Produção e Preservação.

O **AgroForte** é um projeto de front-end conceitual para uma plataforma web focada no agronegócio moderno. O objetivo principal do design é transmitir a força da produção agrícola tradicional em harmonia com as práticas de tecnologia verde e sustentabilidade ambiental para o futuro do planeta.

---

## 🚀 Demonstração Visual

A interface utiliza uma paleta de cores bio-centrada para criar conexões visuais imediatas com o tema:
*   **Verde Escuro (`#1b4332`):** Representa a força da terra e o agronegócio estruturado.
*   **Dourado Ouro (`#d4af37`):** Simboliza a colheita, o sucesso da safra e o valor econômico.
*   **Verde Claro (`#b7efc5`):** Evoca a sustentabilidade, renovação e tecnologia limpa.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído puramente com tecnologias web nativas, garantindo leveza e carregamento rápido:

*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização avançada, uso de CSS Variables (Root) e sistema de Grid/Flexbox Layout.
*   **FontAwesome (v6.4.0):** Ícones vetoriais responsivos para representação visual dos pilares.
*   **Unsplash API:** Imagens conceituais de alta definição integradas diretamente no código.

---

## 📂 Estrutura do Projeto

O repositório está organizado de forma simples e direta:

```bash
├── index.html      # Página principal com a estrutura semântica
└── style.css       # Folha de estilo contendo as regras de design e identidade visual
```

---

## 🎯 Funcionalidades e Seções

1.  **Navegação Fixa (Sticky Navbar):** Menu superior inteligente que acompanha a rolagem do usuário.
2.  **Banner Principal (Hero Section):** Chamada de impacto com imagem de fundo adaptada e botões de ação (CTA).
3.  **Grade de Pilares (Features Grid):** Cards interativos que destacam os valores de Produção Eficiente, Gestão Hídrica e Carbono Neutro com efeitos de *hover*.
4.  **Seção de Destaque Tecnológico:** Espaço dedicado à explicação das inovações de campo (como IA e monitoramento de solo).
5.  **Rodapé Institucional:** Fechamento profissional com assinatura de direitos autorais.

---

## 💻 Como Executar o Projeto

Não é necessário instalar nenhuma dependência ou configurar servidores complexos.

1.  Faça o clone ou baixe os arquivos `index.html` e `style.css`.
2.  Certifique-se de manter os dois arquivos na **mesma pasta**.
3.  Abra o arquivo `index.html` em qualquer navegador web de sua preferência (Chrome, Edge, Firefox, Safari).
